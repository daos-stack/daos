§ cont_iv_prop_fetch()

int cont_iv_prop_fetch ( struct ds_iv_ns *  ns,
uuid_t  cont_hdl_uuid,
daos_prop_t *  cont_prop 
)

Query container properties.

Parameters
[in]nspool IV namespace
[in]cont_hdl_uuidcontainer handle uuid
[out]cont_propreturned container properties If it is NULL, return -DER_INVAL; If cont_prop is non-NULL but its dpp_entries is NULL, will query all pool properties, DAOS internally allocates the needed buffers and assign pointer to dpp_entries. If cont_prop's dpp_nr > 0 and dpp_entries is non-NULL, will query the properties for specific dpe_type(s), DAOS internally allocates the needed buffer for dpe_str or dpe_val_ptr, if the dpe_type with immediate value then will directly assign it to dpe_val. User can free the associated buffer by calling daos_prop_free().
Returns
0 if Success, negative if failed.
