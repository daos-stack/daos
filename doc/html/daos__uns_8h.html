Detailed Description

DAOS Unified Namespace API

The unified namespace API provides functions and tools to be able to link files and directories in a system namespace to a location in the DAOS tier (pool and container), in addition to other properties such as object class.

§ duns_link_path()

int duns_link_path ( const char *  path,
const char *  sysname,
d_rank_list_t *  svcl,
struct duns_attr_t *  attrp 
)

Create a special directory (POSIX) or file (HDF5) depending on the container type, and create a new DAOS container in the pool that is passed in attr.da_puuid. The uuid of the container is returned in attr.da_cuuid. Set extended attributes on the dir/file created that points to pool uuid, container uuid, and default object class to be used for object in that container. This is to be used in a unified namespace solution to be able to map a path in the unified namespace to a location in the DAOS tier.

Parameters
[in]pathValid path in an existing namespace.
[in]sysnameDAOS system name.
[in]svclDAOS pool service ranks list. NOTE: svcl is temporary and will be removed.
