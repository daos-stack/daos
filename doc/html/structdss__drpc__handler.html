Detailed Description

Any dss_module that accepts dRPC communications over the Unix Domain Socket must provide one or more dRPC handler functions. The handler is used by the I/O server to multiplex incoming dRPC messages for processing.

The dRPC messaging module ID is different from the dss_module's ID. A dss_module may handle more than one dRPC module ID.
