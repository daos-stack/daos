◆ daos_progress()

int daos_progress ( tse_sched_t *  sched,
int64_t  timeout,
bool *  is_empty 
)

Make progress on the RPC context associated with the scheduler and schedule tasks that are ready. Also check if the scheduler has any tasks.

Parameters
sched[IN] Scheduler to make progress on.
timeout[IN] How long is caller going to wait (micro-second) if timeout > 0, it can also be DAOS_EQ_NOWAIT, DAOS_EQ_WAIT
is_empty[OUT] flag to indicate whether the scheduler is empty or not.
Returns
0 if Success, negative DER if failed.

◆ daos_task_create()

int daos_task_create ( daos_opc_t  opc,
tse_sched_t *  sched,
unsigned int  num_deps,
tse_task_t *  dep_tasks[],
tse_task_t **  taskp 
)

Create an asynchronous task and associate it with a daos client operation. For synchronous operations please use the specific API for that operation. Typically this API is used for use cases where a list of daos operations need to be queued into the DAOS async engines with specific dependencies for order of execution between those tasks. For example, a user can create task to open an object then update that object with a dependency inserted on the update to the open task. For a simpler workflow, users can use the event based API instead of tasks.

Parameters
opc[IN] Operation Code to identify the daos op to associate with the task,
sched[IN] Scheduler / Engine this task will be added to.
num_deps[IN] Number of tasks this task depends on before it gets scheduled. No tasks can be in progress.
dep_tasks[IN] Array of tasks that new task will wait on completion before it's scheduled.
taskp[OUT] Pointer to task to be created/initialized with the op.
Returns
0 Success -DER_INVAL Invalid parameter -DER_NOSYS Unsupported opc

Reset a DAOS task with another opcode. The task must have been completed or not in the running state yet, and has not been freed yet (use must take a ref count on the task to prevent it to be freed after the DAOS operation has completed).

Parameters

opc[IN] Operation code to identify the daos op to associate with the task.
